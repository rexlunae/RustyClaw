Gateway management (start / stop / restart / status)

Usage: rustyclaw gateway [OPTIONS] <COMMAND>

Commands:
  install    Install gateway as a system service (systemd/launchd)
  uninstall  Uninstall gateway system service
  start      Start the gateway (daemon/background)
  stop       Stop a running gateway
  restart    Restart the gateway
  status     Show gateway status
  reload     Reload gateway configuration without restarting
  logs       Show gateway logs
  run        Run the gateway in the foreground (like `rustyclaw-gateway`)
  help       Print this message or the help of the given subcommand(s)

Options:
  -c, --config <PATH>       Path to a config.toml file [env: RUSTYCLAW_CONFIG=]
      --settings-dir <DIR>  Settings directory (default: ~/.rustyclaw) [env: RUSTYCLAW_SETTINGS_DIR=]
      --profile <PROFILE>   Isolate state under ~/.rustyclaw-<PROFILE> [env: RUSTYCLAW_PROFILE=]
      --no-color            Disable coloured terminal output [env: NO_COLOR=]
      --soul <PATH>         Path to SOUL.md [env: RUSTYCLAW_SOUL=]
      --skills <DIR>        Skills directory [env: RUSTYCLAW_SKILLS=]
      --no-secrets          Disable secrets storage
      --gateway <WS_URL>    Gateway WebSocket URL (ws://â€¦) [env: RUSTYCLAW_GATEWAY=]
  -h, --help                Print help
